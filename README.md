# @alchemiststudios/pi-harness-skills

Harness Engineering skill pack for pi.

## Install

### From local path

```bash
pi install /absolute/path/to/pi-harness-package
```

### From npm

```bash
pi install npm:@alchemiststudios/pi-harness-skills@0.1.0
```

## Included skills

- agents-md-mapper
- ast-grep-setup
- differential-session-runner
- execute-phase
- harness-map
- plan-phase
- qa-from-execute
- research-phase
